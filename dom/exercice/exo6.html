Création d’un chronomètre En partant d’une page html simple avec le bloc suivant :
0 heure(s) : 0 minute(s) : 0 seconde(s) :
Créez un chronomètre à travers une fonction qui sera lancera via le click sur le bouton start et en utilisant le DOM affichera les valeurs au bon endroit Quand on a cliquer sur START, le texte se transform en STOP et si vous cliquez dessus le chronometre s'arrete
